# Arena to Liga

## Como utilizar
#### 1° - exporte a sua lista no MTG Arena
#### 2° - acesse a aplicação no seguinte endereço: https://arena-to-liga.vercel.app
#### 3° - cole a cópia da sua lista na aplicação e aperte o botão para formatar

Agora é só utilizar a lista formatada no site da Liga Magic!
